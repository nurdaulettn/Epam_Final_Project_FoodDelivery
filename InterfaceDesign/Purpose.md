# Цель приложения
Приложение — платформа для онлайн-заказа еды из ресторанов. Пользователь выбирает ресторан и блюда, оформляет заказ и отслеживает его статус. Ресторан получает и обрабатывает заказы, управляет меню. Администратор следит за работой всей платформы.
Ценность для пользователей: быстро и удобно заказать еду из любого ресторана, не нужно звонить или ехать лично.

# Цели пользователей и сценарии
Базовый сценарий (клиент):
Пользователь хочет заказать еду. Открывает сайт → видит список ресторанов → выбирает ресторан → добавляет блюда в корзину → оформляет заказ → оплачивает → отслеживает статус.

Сценарий менеджера:
Открывает панель ресторана → видит новый заказ → принимает его → обновляет статус «Готовится» → ставит «Готов».
